课题二：校园生活百事通 （智能问答知识库）
背景：新生入学或日常学习生活中，学生常需查询校历、食堂开放时间、图书馆规则、
报修流程、社团联系方式等。但信息分散在官网、公众号、学生手册中。本课题旨在构建一
个集中的校园生活知识库，提供便捷的自然语言问答服务。
核心功能：
1） 多源知识导入：支持管理员批量导入常见问题，或爬取/上传校园官网文档。
2） 智能检索问答：用户输入问题（如“食堂几点关门？” “在哪里打印”“校车时刻表”），
系统先进行意图识别与关键词检索，返回最匹配的答案；可接入大模型 API 生成更
友好的回答。
3） 用户贡献机制：允许用户补充新问答，经审核后入库，实现众包更新。
4） 热点问题推荐：基于高频查询词展示“常见问题排行榜”。
